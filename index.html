How to Use

Select cube size (2×2 to 5×5) from the buttons above. Drag to rotate view. Use move buttons or keyboard: R L U D F B (Shift for counter-clockwise). For 4×4+, use wide moves (Rw, Lw, etc.) and inner layers. Scramble randomizes, Solve reverses all moves.
